Опциональный контент-мод для полной русификации статических данных:

- Скопируйте нужные XML из папки игры (StreamingAssets/Base) в подпапку этого мода с сохранением относительных путей, переведите атрибуты текста и подключите мод в списке модов игры.
- Рантайм-мод RuLocalization перехватывает строки через Harmony; большие объёмы текста надёжнее дублировать переведёнными данными здесь, чем полагаться только на машинный перевод.
